# AskMe - Приложение для публикации и обсуждения вопросов

AskMe - это веб-приложение, разработанное на Django, которое позволяет пользователям публиковать вопросы и отвечать на них. Проект представляет собой платформу для обмена знаниями, где пользователи могут задавать вопросы по различным темам и получать ответы от сообщества.

## Функционал приложения

- **Регистрация и авторизация пользователей** - возможность создания аккаунта и входа в систему
- **Публикация вопросов** - создание новых вопросов с заголовками и подробными описаниями
- **Ответы на вопросы** - возможность отвечать на опубликованные вопросы
- **Оценка контента** - система голосования за вопросы и ответы (лайки/дизлайки)
- **Профили пользователей** - персональные страницы с информацией о пользователе и его активности
- **Система тегов** - классификация вопросов по тематическим меткам для удобного поиска
- **Рейтинг пользователей** - система репутации на основе активности и полезности ответов
- **Поиск по вопросам** - возможность находить интересующие темы по ключевым словам

## Установка и запуск

### Локальный запуск:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/pavel-ladygin/django-askme
   cd django-askme
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python3.11 -m venv venv
   source venv/bin/activate  # На Windows: venv\Scripts\activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   pip install psycopg2-binary
   ```

4. Выполните миграции базы данных:
   ```bash
   python manage.py migrate
   ```

5. Создайте суперпользователя (опционально):
   ```bash
   python manage.py createsuperuser
   ```

6. Запустите локальный сервер:
   ```bash
   python manage.py runserver
   ```

   Приложение будет доступно по адресу: http://localhost:8000/

### Запуск через Docker:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/pavel-ladygin/django-askme
   cd django-askme
   ```

2. Запустите докер-контейнеры:
   ```bash
   docker-compose up --build -d
   ```

3. Примените миграции и заполните базу данных тестовыми данными:
   ```bash
   docker-compose exec web python manage.py migrate
   docker-compose exec web python manage.py filldb [ratio]
   ```

   Где `[ratio]` - коэффициент количества генерируемых тестовых данных.

4. После завершения генерации данных откройте приложение:
   - http://localhost:8000/

## Требования

- Python 3.11+
- PostgreSQL (для локальной установки)
- Docker и Docker Compose (для запуска в контейнерах)
